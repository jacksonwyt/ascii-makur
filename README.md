# ASCII Makur

Standalone ASCII editor inspired by ASC11's editor flow.

## Run locally

```bash
npm run dev
```

Then open `http://localhost:4173`.

## What it includes

- image and video upload
- multiple ASCII styles
- dithering controls
- color and FX controls
- mouse interaction
- raster export
- SVG export
- GIF and MP4 export via browser-side recording and FFmpeg wasm

## Deploy

This is a static app. Any static host is fine:

- Netlify
- Cloudflare Pages
- Vercel static output
- GitHub Pages

Serve the repository root as the site root.
